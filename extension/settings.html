ℹ️ About

Org Memory Engine v1.0

A chrome extension that captures organizational decisions and extracts insights using AI.

Features:
✓ Capture text from any webpage
✓ Extract decisions automatically
✓ Ask questions with RAG
✓ Real-time agent monitoring
✓ Store in PostgreSQL

Technology Stack:
• FastAPI (Backend)
• PostgreSQL (Database)
• ChromaDB (Vector Search)
• Google Gemini (LLM)
• Chrome Extension API
